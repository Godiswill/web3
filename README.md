：发起一笔币币交易，可用任意钱包转账 A 币到池子地址，它会自动把你想要的 B 币种（可以跨链。有的时候币 OKX 的跨链桥便宜）转到你提供的钱包地址里。（执行过几次小笔交易，几分钟内到账）
